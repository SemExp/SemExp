# SemExp: Multi-Turn Semantic Exploration for Analytical Querying over Hybrid Textual Tables

This repository contains the experimental code for the paper submission:

**SemExp: Multi-Turn Semantic Exploration for Analytical Querying over Hybrid Textual Tables**

## Overview

SemExp is designed for multi-turn semantic exploration and analytical querying over hybrid textual tables. This codebase includes the implementation and evaluation scripts used in the experiments.

## Configuration

Before running the code, please configure the required settings in the source code, such as dataset paths, model settings, API keys, and output directories.

Make sure these configurations are correctly set according to your local environment.

## Usage

### Run the Main Program

After completing the necessary configuration, you can run the main pipeline with:
